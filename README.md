# CQ Satellites

A Swift-based iOS application designed for Amateur Radio (HAM) enthusiasts to quickly find the next available satellite pass.

## About

CQ Satellites simplifies satellite tracking for HAM radio operators by providing real-time orbital predictions and pass timing. Whether you're communicating via amateur radio satellites or coordinating with other operators, CQ Satellites helps you identify upcoming satellite passes so you never miss a transmission opportunity. Built specifically with the needs of the amateur radio community in mind.

## Getting Started

### Prerequisites

- iOS 14.0 or later
- Xcode 12.0 or later
- Swift 5.3 or later

### Installation

1. Clone the repository:
```bash
git clone https://github.com/m0hdy/cq-satellites.git
cd cq-satellites
```

2. Open the project:
```bash
open CQSatellites.xcodeproj
```

3. Build and run the application in Xcode

## Features

- **All Upcoming Passes** - View a complete list of upcoming satellite passes, not just the next one
- **Live Countdown Timer** - See exactly how long until the next pass with a real-time countdown
- **Amateur Radio Satellite Support** - Track popular HAM radio satellites (ISS, SO-50, FM-4, and more)
- **Real-time Orbital Mechanics** - Accurate pass predictions using current TLE data
- **Interactive Visualizations** - Visualize satellite trajectories and ground coverage
- **Customizable Tracking** - Filter satellites by frequency, mode, and pass quality
- **Quick Reference** - Get essential details: rise time, peak elevation, set time, and doppler shift
- **Be Ready** - Never miss an opportunity with advance notice and timely alerts
