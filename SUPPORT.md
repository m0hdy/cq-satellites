# Support

## Where to ask for help

Use the channel that best matches your question:

- **Bug reports:** open a GitHub issue and include repro steps, iOS version, device/simulator details, and logs/screenshots when possible
- **Feature requests:** open a GitHub issue describing the operator problem you want solved
- **Questions / usage help:** use GitHub Discussions if enabled for this repository; otherwise use the question issue template
- **Security concerns:** follow [SECURITY.md](SECURITY.md) and do not post vulnerabilities publicly

## Before opening a support request

Please check:

- [README.md](README.md) for setup, testing, and platform expectations
- [CONTRIBUTING.md](CONTRIBUTING.md) for contributor workflow and common failure modes
- [docs/ARCHITECTURE.md](docs/ARCHITECTURE.md) for project structure and technical context

## Maintainer expectations

This project is maintained on a best-effort basis. Response times may vary, but clear reports with reproduction details are much easier to triage.
